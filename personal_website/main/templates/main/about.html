HTML

HTML is the standard programming language for creating web (internet) pages. HTML stands for Hyper Text Markup Language (Hyper Text Markup Language) to code and write a programming language that describes the external structure of web pages, describes the building blocks of web pages, marks web page elements with HTML tags, and labels parts of a page's content, such as headings, paragraphs, tables, and more. * Web browsers do not display HTML tags, but use HTML to display page content.

CSS

It is the language that is shaped by the aesthetic appearance of the pages, the height and width of the elements in the web page, and controls the layout of the page, so CSS controls the design elements like HTML CSS transforms solid web pages with static content into vibrant pages by giving an aesthetic appearance to page elements using colors, spacing, fonts, and animated elements. In short, CSS identifies some HTML elements and then selects some properties that can be applied to these elements. Each property has value be determined.

Django

Django is a high-level Python web framework that enables rapid development of maintainable websites. A framework is nothing more than a set of modules that make development easier. The news source is consistent, from writing the code from scratch. Django is free and open source (open source), diverse in a thriving and active community, great documentation (documentation), many options for free and paid support. In simple terms, Django is a collection of ready-made components that help you quickly build websites with a functional and clean architecture. When designing websites, developers often need the same components for tasks like authenticating users, uploading files, implementing content management systems, etc.
